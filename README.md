# twitter-clone

Fully functional mern-stack twitter clone application

## Used technologies
 - `React`
 - `Redux`
 - `Tailwind`
 - `Axios`
 - `Express`  
 - `Mongodb`  
 - `Multer`  
 - `Sharp`  
 - `JWT`
